# RETUSSについて

__RETUSS__ (Real-time Ensure Traceability between UML and Source-code System) は、UMLとソースコード間のトレーサビリティをリアルタイムに維持するツールです。

## 言い訳

現在は何もありませんが、これから少しずつ大きくしていこうと思います。

## 開発環境

* Windows 10 Pro (64 bit) 1703
* Java9
* JUnit5 (5.0.1)
